# FinancialFraudDetection

## Цель проекта
Цель проекта заключается в применении разных ML-алгоритмов для решения задачи детектирования признаков мошенничества (фрода) в финансовых данных с целью сравнения качества обнаружения и определения субоптимального алгоритма

## Используемые ML-алгоритмы
- Isolation Forest (Изоляционный лес)
- Random Forest (Случайный лес)
- Linear Regression (Линейная регрессия)
- ...

## Технологический стэк проекта
- Python

## Источники наборов данных для исследования
- Financial Transactions Dataset: Analytics - https://www.kaggle.com/datasets/computingvictor/transactions-fraud-datasets
- Synthetic Financial Datasets For Fraud Detection (PaySim) - https://www.kaggle.com/datasets/ealaxi/paysim1
- Libra Bank transaction graph over 3 months - https://graphomaly.upb.ro/#datasets
